DEVELOPMENT BUILDS

I make a new build of the game every time I think theres a substantial enough of an improvement from the last one.

(I would have the actual showcases if there wasn't a size limit on github)




FIRST BUILD

April 21, 2024

Despite developmenent starting a month prior, this was the very first build of the game. Mostly was created to make sure I hadn't screwed up the project settings (like for my last project). It included a rough stylized start screen, and the basics of all games. The game had a pixelation shader applied to everything, that is now being used for other cases.

SECOND BUILD

April 28, 2024

Showcasing the earliest form of an actual game, with a minigame. Improved the experimental start screen. There was still not a clear enough direction of a main gameplay loop at this time, but there was a very bare-bones FPS system.

THIRD BUILD

May 19, 2024

This build did have a showcase, but was never released because I wanted to combine it with a later build. It really only added a landing sequence, launching sequence, and an imersive orbiting sequence. Along with a custom VHS shader for the game. I experimented with a HUD rendered entirely in 3D space. I have since reverted back to a typical 2D user interface.

FOURTH BUILD

June 12, 2024

Mostly focused on a super rough and early intro mission for the game. A mission that has completely been scraped as of writing this. Also added the first FPS animations, and a better FPS system (still basic).

FIFTH BUILD

July 29, 2024

Focused around an in-depth space combat system, lore cutscenes, and heavy refinement of that intro mission. I also gave alot more depth to the FPS system by adding secondary weapons (grenades and throwing knives), levels and skills, and inventory systems.
