BUILDS

I make a new build of the game every time I think theres a substantial enough of an improvement from the last one.

(I would have the actual showcases if there wasn't a size limit on github)




First Build

April 21, 2024

First build was really just making me sure I could build the game and hadn't broken anything (like for my last project). Also super early experimental start screen.


Second Build

April 28, 2024

Showcasing the earliest form of an actual game, with a minigame. Improved the experimental start screen. There was still not a clear enough direction of a main gameplay loop at this time.



Third Build (fakeh)

May 19, 2024

This build did have a showcase, but was never released because I wanted to combine it with a later build. It really only added a landing sequence, launching sequence, and an imersive orbiting sequence. Along with a custom VHS shader for the game.



Third Build (reahl)

June 12, 2024

Mostly focused on a super rough and early intro mission for the game. A mission that has completely been scraped as of writing this. Also added the first FPS animations, and a better FPS system (still basic).



Fourth Build

July 29, 2024

Focused around an in-depth space combat system, along with lore cutscenes, heavy refinement of that intro mission, and giving more depth to the FPS system.
